## 平台简介

PaiChi(拍吃)，让人们在宣泄的都市中体验在家常做菜，享受美食的乐趣。

2.  登录、注册、找回密码：<br>

    (2)找回密码通过邮箱，发送到邮箱验证链接，用户点击链接完成找回密码验证。<br>
